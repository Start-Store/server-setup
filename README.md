# 🚀 FiveM Server Setup 

Ce serveur est basé sur la configuration de base FiveM avec des ressources custom et natives, sans ESX, QBCore ou autre framework pour deployer sur votre hébérgeur,vps où autre.

---

## 📦 Prérequis

Avant installation, assure-toi d’avoir :

- FXServer artifacts : https://runtime.fivem.net/
- Git installé : https://git-scm.com/
- MariaDB / MySQL (si base de données utilisée)
- Node.js (si scripts JS présents)
- Un VPS ou serveur dédié

---

## ⚙️ Installation

### 1. Télécharger FXServer

Télécharge et installe les artifacts FXServer :
https://runtime.fivem.net/artifacts/fivem/

Décompresse dans un dossier serveur :

C:\FXServer\

---

### 2. Installer le serveur
